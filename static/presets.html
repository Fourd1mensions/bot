Syntax Guide
Placeholders
{key} Replace with value. Unknown keys stay as literal text.
Conditionals
{?key}...{/key} Show block if key is non-empty
{!key}...{/key} Show block if key is empty (negation)
{?key}...{:key}...{/key} If / else — first branch if true, second if false
Comparisons
{?key=val}equals {?key!=val}not equals {?key>val}greater than {?key<val}less than {?key>=val}greater or equal {?key<=val}less or equal
All comparisons support {:key} else clause. Use {! prefix to negate any condition.
Loops
{#array}...{/array} Repeat block for each item in array (e.g., scores)
{.field} Access field of current item inside loop
{#index} Current index (0-based: 0, 1, 2...)
{#position} Current position (1-based: 1, 2, 3...)
Inside loops, use {.field} for per-item values and {field} for global values.
Colors
#rrggbb Hex color (e.g., #ff66aa, #7c4dff)
viola_purple#7c4dff osu_pink#ff66aa easy#4290f5 normal#4fc0ff hard#ffb641 insane#ff6682 expert#c967e5 success#2ecc71 error#ff4444 warning#f1c40f
Examples
Pass / Fail with else
{rank}{?completion} ({completion}){:completion} FC{/completion}
Pass: S FC Fail: F (85.32%)
FC suggestion
**{pp}PP**{?fc_line} {fc_line}{/fc_line}
FC: **350PP** Miss: **300PP** (350pp for 99% FC)
Zero misses = FC
{?miss=0}Full Combo!{:miss}{miss} miss(es){/miss}
0: Full Combo! 3: 3 miss(es)
PP threshold
{?pp>500}🔥 {/pp}**{pp}PP**
300pp: **300PP** 600pp: 🔥 **600PP**
Rank detection
{?rank_raw=SS}⭐ PERFECT{:rank_raw}{rank}{/rank_raw}
SS: ⭐ PERFECT A: (rank emoji)
High accuracy
{?acc>=99.50}Insane acc! {/acc}{acc}%
99.73%: Insane acc! 99.73% 95%: 95.00%
Negation
{!completion}✓ Passed{/completion}
Pass: ✓ Passed Fail: (hidden)
Try counter
{?try_line}{try_line} — {/try_line}{rank} **{pp}PP**
No: S **350PP** Yes: Try #5 — S **350PP**
Score loop
{#scores}#{#position} {.username} **{.pp}pp**\n{/scores}
#1 cookiezi **800pp** #2 Rafis **750pp**
